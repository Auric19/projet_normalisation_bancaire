PROJET D’APPLICATION NORMALISATION BANCAIRE



PARTIE STORAGE : 

Etape 1 : Aller sur le site de postgreSql

Etape 2 : Télécharger la dernière version de postgreSql

Etape 3 : Ouvrir pgAdmin4 qui s’est téléchargé en même temps que postgreSql

Etape 4 : Créer un serveur « NormalisationBancaire »

Etape 5 : Créer une Database « postgres » dans le serveur « NormalisationBancaire » avec le mot de passe de votre choix (il faudra renseigner le même dans « launcher.py »)



PARTIE PROCESSING :

Etape 1 : Télécharger les fichiers du GitHub

Etape 2 : Ouvrir les fichiers dans un IDE

Etape 3 : Installer pySpark sur son terminal

Etape 4 : Run le programme launcher.py



PARTIE VISUALISATION : 

Etape 1 : Aller sur le site de Metabase

Etape 2 : Télécharger MetabaseDesktop

Etape 3 : Ouvrir Metabase

Etape 4 : Renseigner les mêmes informations que dans pgAdmin4

Etape 5 : Se connecter et visualiser les graphes
